# 기능 명세서

## 1. 총괄 관리자 (SUPER_ADMIN)

- 매장 생성 / 수정 / 삭제
- 매장 목록 조회
- 관리자 계정 생성
- 시즌 생성 및 종료
- 전체 시즌 목록 조회
- 전체 랭킹 조회
- 매장별 통계 조회
- 전체 디바이스 상태 조회

---

## 2. 매장 관리자 (STORE_ADMIN)

- 소속 매장 정보 조회
- 소속 매장 랭킹 조회
- 소속 매장 기기 등록
- 소속 매장 기기 수정
- 소속 매장 기기 상태 조회

---

## 3. 매칭 시스템

- 매치 기록 저장 API 제공
- UUID 기반 외부 식별자 관리
- 시즌 자동 매핑 (날짜 기반)
- 맵 정보 기반 기록 저장

---

## 4. 랭킹 시스템

- 전체 랭킹 조회
- 시즌별 랭킹 조회
- 매장 내 랭킹 조회
- 개인 랭킹 조회 (닉네임 검색)

---

## 5. 비기능 요구사항

- 모든 PK는 BIGINT 사용
- 모든 FK는 참조 무결성 유지
- 시즌 기간 중복 금지
- ENUM은 고정 속성에만 사용
- 랭킹 조회 성능을 위한 인덱스 최적화
- UUID는 외부 공개용 식별자로 사용

---

## 6. v2 예정 기능

### 6.1 디바이스 시스템

- 매장별 기기 등록 및 상태 관리
- 기기 마지막 통신 시간 추적
- 오프라인 기기 감지
- 통합 관리자 대시보드 연동

### 6.2 장부 시스템 (운영 확장 기능)

- 매장별 일간 매출 집계
- 매장별 월간 매출 집계
- 시즌 기준 매출 분석
- 총 플레이 횟수 집계
- 맵(대/중/소)별 매출 통계
